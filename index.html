My Projects

Kannada Paraphrase Generation (KnParaphraser)

Developed a Kannada paraphrasing model Kn-T5 using data augmentation. Created and released KanPara (100k sentence corpus) using backtranslation with NLLB-200. Published on Springer and Hugging Face. Achieved BLEU score: 0.1967, similarity: 0.8133.

Neural Architecture Search for Image Deblurring:

Designed genetic encoding, decoding, crossover (Uniform + SBX), and mutation operators. Used NSGA-II to balance PSNR and FLOPs. Currently evaluating on the GoPro dataset.

Women Employment Web Portal:

Built a full-stack AI-powered job portal with Node.js and OpenAI. Implemented NLP-based job matching using LinkedIn API. Improved accuracy by 30%, tested with Selenium, and optimized backend speed by 30%.

Knee Arthritis Detection using X-rays with Explainable AI (XAI):

Used OAI dataset with 9,786 X-rays. Developed KL grade classifier. Integrated Grad-CAM for interpretability. Working on quantization and mobile deployment with TensorFlow Lite.

Neural Machine Translation for Kannada-to-Dravidian Languages

Built high-quality parallel corpora for Kannada-Tamil, Kannada-Telugu, and Kannada-Malayalam translations. Applied IndicSBERT filtering, and fine-tuned mT5 for accurate multilingual translations. Contributing to low-resource NLP research.

Tourist City Management using DSA

Implemented graph algorithms (Dijkstra’s, DFS, Prim’s) for transport and emergency services. Designed data structures to manage city layouts, commercial hubs, and attractions. Developed a parking and taxi dispatch system with Rabin-Karp for efficient text-based search.
